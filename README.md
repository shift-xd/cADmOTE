
# cADmOTE is a 3d accelormeter powered remote to rotate , manupulate and view your 3d models in all cad !
## cADmOTE is one of my creations for HACK CLUB BLUEPRINT event 
### Cadmote uses an xiao Rp2040 board an accelrometer and 5 buttons you can program
